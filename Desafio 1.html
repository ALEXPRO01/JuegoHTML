Bienvenido al Desafío 1

Crear la estructura básica de un documento HTML

En este desafío, tu tarea es crear la estructura básica de un documento HTML.

  • Primero, crea una carpeta para tu página web, dentro de esta carpeta vas a crear un documento de texto y a este documento lo vas a nombrar de la siguiente manera ('index.html').

    • Para abrir esta carpeta en Visual Studio Code, haz lo siguiente:

  • Entra al editor de texto, en este caso es Visual Studio Code, dirígete a la parte superior izquierda y podras visualizar la opción ('File') dentro de este apartado aparecerán varias opciones, selecciona la opción ('Open Folfer'), se abrira el explorador de archivos, busca la carpeta que creaste anteriormente y das a la opción seleccionar, tal y como se muestra a continuación.



    • Una vez hayas abierto tu carpeta en el editor de texto vas a procerder a crear la esturctura básica para tu página web.

  • Para hacer esto selecciona el archivo 'index.html' y una vez dentro escribe el siguiente código: html, apareceran tres opciones, selecciona la opción html:5, este codigo creará toda la esturctura básica del documento HTML.

  • Puedes añadir más contenido dentro del cuerpo <body> de tu página, como puede ser un <h1> que sirve para identificar el tema de tu página web. Tambien puedes agregar la etiqueta <p>, se utiliza para representar un párrafo de texto y separar bloques de texto.

  • Para guardar el avance que tienes utiliza la combinación de teclas ctrl+s

    • A continuación, se muestra un ejemplo básico de la estructura de un documento HTML:

           
    
    
    <!DOCTYPE html> 
    
    <html lang="es"> 
    
    <head> 
    
        <meta charset="UTF-8"> 
    
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
    
        <title>Mi Primer Documento HTML</title> 
    
    </head> 
    
    <body> 
    
        <h1>Hola mundo</h1> 
    
        <p>Este es mi primer documento HTML.</p> 
    
    </body>
    
    </html>

    Una vez que hayas creado la estructura básica del documento, puedes pasar al siguiente desafío.

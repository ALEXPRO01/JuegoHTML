Bienvenido al Desafío 7

Tablas


En este desafío, aprenderás a crear tablas en HTML y a aplicar estilos CSS para mejorarlas visualmente. Las tablas son útiles para organizar y presentar datos de forma clara y estructurada.


Instrucciones:


  1. Crea una tabla utilizando la etiqueta <table>.

  2. Dentro de la tabla, define una fila de encabezado con la etiqueta <tr> y dentro de ella, las celdas de encabezado con la etiqueta <th>.

  3. Añade varias filas de datos utilizando la etiqueta <tr> y dentro de ellas, las celdas de datos con la etiqueta <td>.

  4. Aplica estilos CSS para mejorar la apariencia de la tabla. Utiliza propiedades como border, padding y background-color.

Ejemplo:


Sigue este ejemplo para crear tu propia tabla:


                

                
                    <table>
                        <tr>
                            <th>Nombre</th>
                            <th>Edad</th>
                            <th>Ciudad</th>
                        </tr>
                        <tr>
                            <td>Juan</td>
                            <td>25</td>
                            <td>Quito</td>
                        </tr>
                        <tr>
                            <td>María</td>
                            <td>30</td>
                            <td>Guayaquil</td>
                        </tr>
                        <tr>
                            <td>Carlos</td>
                            <td>28</td>
                            <td>Loja</td>
                        </tr>
                    </table>

Intenta crear tu propia tabla siguiendo los pasos anteriores y utilizando el ejemplo como guía. Buena suerte.

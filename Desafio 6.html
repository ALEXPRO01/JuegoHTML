Bienvenido al Desafío 6

Listas


En este desafío, aprenderás a crear listas en HTML. Las listas son útiles para organizar información en un formato estructurado y fácil de leer.


Instrucciones:


  1. Crea una lista ordenada (numerada) utilizando la etiqueta <ol>.
  2. Añade elementos a la lista ordenada utilizando la etiqueta <li> para cada elemento.
  3. Crea una lista desordenada (con viñetas) utilizando la etiqueta <ul>.
  4. Añade elementos a la lista desordenada utilizando la etiqueta <li> para cada elemento.
  5. Aplica estilos CSS para mejorar la apariencia de las listas, utilizando propiedades como margin, padding y background-color.

Ejemplo:


Sigue este ejemplo para crear tus propias listas:


Lista Ordenada

                
                    <ol>
                        <li>Primer elemento</li>
                        <li>Segundo elemento</li>
                        <li>Tercer elemento</li>
                    </ol>

Lista Desordenada

                
                    <ul>
                        <li>Primer elemento</li>
                        <li>Segundo elemento</li>
                        <li>Tercer elemento</li>
                    </ul>

Intenta crear tus propias listas siguiendo los pasos anteriores y utilizando el ejemplo como guía. Buena suerte.

Etiquetas Semánticas

En este desafío, aprenderás sobre la importancia de las etiquetas semánticas en HTML y cómo utilizarlas para mejorar la estructura y la accesibilidad de tus páginas web.

Las etiquetas semánticas son elementos HTML que proporcionan información sobre el contenido que encierran. Al utilizar estas etiquetas, estás comunicando de manera clara la estructura y el significado de tu contenido tanto para los usuarios como para los motores de búsqueda.

Las etiquetas semánticas ayudan a mejorar la accesibilidad de tu sitio web, permitiendo que los usuarios con discapacidades visuales o lectores de pantalla puedan entender mejor la estructura de la página. Además, los motores de búsqueda utilizan esta información para indexar y clasificar tu contenido de manera más precisa.

Algunas de las etiquetas semánticas más comunes son:

  • <header>: Define el encabezado de una página o sección.
  • <nav>: Define una sección de navegación.
  • <main>: Define el contenido principal del documento.
  • <article>: Define contenido independiente y relevante, como artículos o publicaciones.
  • <section>: Define una sección genérica de contenido.
  • <aside>: Define contenido relacionado pero separado del contenido principal, como barras laterales.
  • <footer>: Define el pie de página de una página o sección.

Ahora es tu turno. Utiliza las etiquetas semánticas para estructurar la siguiente página web.

Instrucciones

1. Crea una estructura básica de HTML con las etiquetas <!DOCTYPE html>, <html>, <head>, y <body>.

2. Añade un <header> para el encabezado de la página.

3. Crea una barra de navegación utilizando la etiqueta <nav>.

4. Define el contenido principal con la etiqueta <main>.

5. Dentro del <main>, añade un <section> y un <article> para estructurar el contenido.

6. Añade un <aside> para contenido adicional, como enlaces o publicidad.

7. Finalmente, añade un <footer> para el pie de página.

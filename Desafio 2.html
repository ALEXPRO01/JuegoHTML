Diseño de Pagina Web con estilos CSS


Instrucciones para vincular un archivo CSS a HTML


Para vincular un archivo CSS externo a un documento HTML, debes seguir los siguientes pasos:


  1. Crea un archivo CSS (por ejemplo, estilos.css) con las reglas de estilo que desees aplicar.

  2. En tu documento HTML, añade la etiqueta <link> dentro del <head>, especificando el atributo href con la ruta al archivo CSS y el atributo rel="stylesheet".

Ejemplo:


<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Página Web</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <!-- Contenido de la página -->
</body>
</html>

A continuación, te mostramos una tabla con algunas de las propiedades CSS más comunes:


Propiedad Descripción
color Define el color del texto.
background-color Establece el color de fondo de un elemento.
font-size Especifica el tamaño de la fuente del texto.
margin Define el margen alrededor de un elemento.
padding Define el espacio interior entre el contenido de un elemento y su borde.

Recuerda que también puedes utilizar clases y selectores para aplicar estilos a elementos específicos o grupos de elementos en tu documento HTML. Aquí tienes un ejemplo de cómo usar clases y selectores en CSS:


/* Archivo CSS: estilos.css */
h1 {
  color: #007bff;
}

.parrafo-destacado {
  font-size: 18px;
  color: #333;
  background-color: #f0f0f0;
  padding: 10px;
  border-left: 5px solid #007bff;
}

/* Archivo HTML */
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo de Clases y Selectores</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <h1>Este es un encabezado</h1>
  <p class="parrafo-destacado">Este es un párrafo destacado con estilos CSS aplicados.</p>
</body>
</html>

Guía de Estilos CSS para Mejorar tu Página Web

A continuación se presentan algunas propiedades CSS adicionales que pueden mejorar la apariencia de tu página web:

  • text-align: Ajusta la alineación del texto. Valores comunes: left, right, center, justify.

  • border: Añade un borde alrededor de un elemento. Ejemplo: border: 1px solid #000;.

  • border-radius: Redondea las esquinas de un elemento. Ejemplo: border-radius: 5px;.

  • box-shadow: Añade una sombra a un elemento. Ejemplo: box-shadow: 0 4px 8px rgba(0,0,0,0.1);.

  • font-family: Cambia la fuente del texto. Ejemplo: font-family: 'Arial', sans-serif;.

  • width y height: Especifica el ancho y alto de un elemento. Ejemplo: width: 100px;, height: 50px;.

  • display: Controla cómo se muestra un elemento. Valores comunes: block, inline, inline-block, flex.

  • flex: Propiedad para hacer diseños flexibles. Ejemplo: display: flex;, justify-content: center;, align-items: center;.

  • position: Controla la posición de un elemento. Valores comunes: static, relative, absolute, fixed, sticky.

Experimenta con estas propiedades y ajusta los valores para ver cómo cambian la apariencia de tu página web.

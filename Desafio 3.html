Imágenes y Enlaces


En este desafío, aprenderás a agregar imágenes y enlaces a tus páginas HTML.


  • Descarga una imagen que te guste o si ya la tienes mueve esa imagen a la carpeta que creaste para tu página web.
  • Dirígete al editor de texto, para insertar la imagen en tu página web, utiliza la etiqueta <img>.


    • Esta etiqueta tiene dos atributos principales:


    • src: Especifica la ruta de la imagen que deseas mostrar.
    • alt: Proporciona un texto alternativo para la imagen, útil para la accesibilidad y en caso de que la imagen no se cargue.

                      <img src="ruta/de/tu/imagen.jpg"alt="Descripción de la imagen">

    Creación de Enlaces

    Para crear un enlace, utilizamos la etiqueta <a>. Esta etiqueta tiene un atributo principal:


    • href: Especifica la URL del destino al que lleva el enlace.

                      <a href="https://www.ejemplo.com">Visita Ejemplo</a>

    Aquí un ejemplo de enlace:


    https://juanxxiii.edu.ec/

    Ejercicio Práctico


    Ahora es tu turno. Inserta una imagen en tu página web y crea un enlace que lleve a un sitio web de tu elección.


    Instrucciones


    1. Encuentra una imagen que te guste.

    2. Inserta la imagen en el lugar adecuado utilizando la etiqueta <img> con los atributos src y alt.

    3. En css puedes ajustar el tamaño de tu imagen con los estilos width y height , estos estilos se miden en pixeles (px) o en porcentaje (%). Ejemplo: width: 20px; /width: 20%; --- height: 20%; / height: 20px%

    3. Crea un enlace a un sitio web de tu elección utilizando la etiqueta <a> con el atributo href.

Enlaces y Navegación


En este desafío, aprenderás a crear enlaces en HTML y a organizar la navegación de tu página web. Los enlaces son fundamentales para conectar diferentes secciones y páginas entre sí, facilitando la experiencia del usuario.


Instrucciones:


  1. Crea un menú de navegación utilizando la etiqueta <nav> y añade enlaces con la etiqueta <a> a diferentes secciones de la misma página.

  2. Utiliza atributos de enlace como href para especificar la ubicación del enlace. Para enlaces internos, utiliza anclas (#) para vincular a elementos con identificadores únicos.

  3. Añade tres secciones en la página con la etiqueta <section> y asigna identificadores únicos a cada una.

  4. Aplica estilos CSS para mejorar la apariencia de los enlaces y la navegación.

Ejemplo:


Sigue este ejemplo para crear tu propio sistema de navegación:


Menú de Navegación:

                
                    <nav>
                        <a href="#seccion1">Sección 1</a>
                        <a href="#seccion2">Sección 2</a>
                        <a href="#seccion3">Sección 3</a>
                    </nav>

Secciones:

                
                    <section id="seccion1">
                        <h3>Sección 1</h3>
                        <p>Contenido de la Sección 1.</p>
                    </section>
    
                    <section id="seccion2">
                        <h3>Sección 2</h3>
                        <p>Contenido de la Sección 2.</p>
                    </section>
    
                    <section id="seccion3">
                        <h3>Sección 3</h3>
                        <p>Contenido de la Sección 3.</p>
                    </section>

Intenta crear tu propio sistema de navegación siguiendo los pasos anteriores y utilizando el ejemplo como guía. Buena suerte.
